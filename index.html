Défi Intensif 3 Jours

Programme complet sur 3 jours avec suivi personnalisé

🔥 JOUR 1 - FONDATIONS

Full Body Intensive #1

Séance complète pour activer tout le corps

  • Squat, Développé couché, Rowing barre : 3x10
  • Curl biceps, Extensions triceps : 3x10
  • Soulevé de terre, Fentes, Tirage vertical : 3x10
  • Gainage, Crunchs : 3x10
  • Variante machines guidées : 3x10
Repos 1'30 entre séries
⚡ JOUR 2 - PERFORMANCE

Full Body Intensive #2

Ciblage musculaire et endurance

  • Presse à cuisses (Quadriceps) : 3x12
  • Tirage vertical (Dos) : 3x12
  • Développé assis (Pectoraux) : 3x12
  • Leg curl (Ischio) : 3x12
  • Écarté machine (Pectoraux/Épaules) : 3x12
  • Gainage planche (Abdos) : 3x45 sec
Repos 1'30 entre séries
💪 JOUR 3 - ELITE

Full Body Intensive #3

Séance avancée pour résultats maximaux

  • Développé couché : 4x8-12
  • Tractions poulie haute : 4x8-20
  • Squat : 4x8-12
  • Glute Ham Raise : 3x12-20
  • Développé militaire : 3x8-15
  • Oiseaux haltères : 3x12-30
  • Curl barre : 3x8-15
  • Barre au front : 3x10-20
  • Crunch Abmat : 4x20-50
Intensité maximale
